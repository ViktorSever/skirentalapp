# BookingApp - Горнолыжное оборудование

Этот проект написан на **KMP**.

## Суть проекта

Мобильное приложение для бронирования горнолыжного оборудования.

## Применяемые технологии:

1. **KMP** (Kotlin Multiplatform Mobile)
2. **Jetpack Compose**
3. **Firebase Firestore**

## Возможности:

1. ✅ Авторизация/Регистрация пользователя
2. ✅ Бронирование оборудования
3. ✅ Просмотр текущих броней

## Быстрый старт

Чтобы посмотреть приложение, необходимо на **Android устройство/эмулятор** установить файл *
*`for-test.apk`**.
